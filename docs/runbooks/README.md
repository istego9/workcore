# Runbooks

Runbooks in this folder must be operationally actionable.

Each runbook should include:
1) Symptoms and blast radius
2) Health checks and exact commands
3) Key logs/metrics to inspect
4) Common root causes
5) Remediation and rollback steps
6) Escalation criteria

Recommended initial runbooks:
- `orchestrator-runtime.md`
- `run-debug-inspector.md`
- `chatkit-integration.md`
- `streaming-sse.md`
- `webhooks-delivery.md`
